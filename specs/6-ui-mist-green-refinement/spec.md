# Mist Green UI – Frontend Visual Refinement Specification

## Objective
Refine the existing Todo App frontend UI using a calm, modern Mist Green theme
without changing any functionality, logic, routing, or backend integration.

## Scope
- Visual styling only (colors, spacing, typography, layout polish)
- Apply a consistent Mist Green theme across all frontend pages
- Ensure UI consistency between Auth pages and main app pages

## In Scope
- Navbar styling
- Login & Register page UI consistency
- Task page layout and card styling
- Button styles, hover & focus states
- Spacing, alignment, border radius, shadows

## Out of Scope
- Any backend changes
- API logic or endpoints
- State management logic
- File renaming or restructuring
- Feature additions or removals

## Theme Definition
- Primary: #8FBFB3 (Mist Green)
- Background: #F4FAF8 (Light Mint / Off-white)
- Accent: #3F6F68 (Dark Sage)
- Text Primary: #1F2F2C
- Text Secondary: #5F7F78
- Error: Muted coral red (non-bright)

## Design Rules
- Minimal, calm, professional look
- Rounded corners (8–12px)
- Soft shadows only
- Smooth transitions (no flashy animations)

## Success Criteria
- UI updated without breaking any existing functionality
- Same theme visible on all pages
- No console errors or UI regressions
- App behavior remains unchanged
